# iPhotography-Passport-Photo-Maker
A professional web-based passport photo maker for creating, editing, printing, and exporting passport and visa photos for multiple countries. Built with HTML, CSS, JavaScript, and image processing tools.

## Features

- Passport photo cropping
- Automatic background removal
- AI image enhancement
- Multiple passport size templates
- Visa photo support
- Print layout generator
- High-resolution export
- Responsive web interface

## Technologies

- HTML5
- CSS3
- JavaScript
- Canvas API
- Image Processing
